My Projects

Personalised Secure Gesture Controlled GUI Automation System

Built a personalized secure gesture controlled GUI automation system. It is an interactive interface that interacts with the user with voice notes. It is provided with a two-step security system using facial recognition and gesture controls. It uses a face detection algorithm which uses the concept of template matching to provide access to specific people. On recognising the person, it directs you to the login page where upon giving where we can navigate through the website by using gestures.
Tools used: OpenCV, Chroma-keying, Contour detection, Template Matching, PyAutoGUI library, gTTS(Google Text to Speech). link
